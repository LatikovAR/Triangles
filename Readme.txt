Этот проект предназначен для поиска пересечений между треугольниками в пространстве и их визуализации.

На данный момент визуализация сделана только под Windows (под Linux можно получить лишь статичную картинку).

Входные данные подаются на стандартный вход в виде числа треугольников и координат их вершин по порядку.

Вырожденные в точку/отрезок треугольники учитываются при поиске пересечений но не визуализруются.

Управление камерой в визуализации:
Перемещение вокруг сцены - Cursor control keys.
Приблизить/отдалить - PageDown/PageUp.
